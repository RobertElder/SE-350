Directory Contents: 

process.h / process.c - contains process related structures, such as pcb and init table entries, as well as process managment related functions, such as initialization  of processes and scheduling.

rtx.h / rtx.c - contains routines for requesting and allocating memory blocks

usr_proc.h / usr_proc.c - contains the 6 test user processes that we created

utils.h / utils.c - contains a number of utility functions, such as random number generator

main_svc.c - contains our main function, initiates processes and memory then runs the processes
